This section outlines the **required software components and dependencies** needed to install, configure, and run the Bull system successfully.

Ensuring that the correct software versions are installed helps maintain compatibility, security, and stability across environments. It also reduces setup errors and ensures smooth integration with supporting tools and services during development and deployment.


### List of Requirements 

1. Python 3.x
2. Docker 
3. Docker Compose (v2 Recommended)
4. Multicast network enabled 
5. PostgreSQL with TimeaScaleDB (manual Installation)
6. Putty (accesing the CMD)
7. WinSCP (for trasferring the files)
